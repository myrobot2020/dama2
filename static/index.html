Dama RAG (local)

Ask questions against your transcript index. Chats are saved per session. Use “Rebuild Index” if you add or change transcript files.
Ollama speed sends one minimal prompt to your local Ollama API (localhost:11434) and shows how many seconds that took — useful to see if the LLM is reachable. Generated answers need Ollama running, the model pulled (e.g. ollama pull mistral:instruct or set env OLLAMA_MODEL), and use Ollama LLM checked. The server log lines about BertModel / Loading weights are the embedding and rerank models (Hugging Face), not Ollama.
